isoTar - Consensus Target Prediction with Functional Enrichment Analysis for Human miRNA Variants


Run on any system, no worries!

isoTar is a high performance web-based containerized application designed for Human miRNA variants. It provides:

  • Consensus Target Prediction
  • Functional Enrichment Analysis
  • Easy download and run (powered by Docker)

Through a friendly web-based User Interface (UI), users can explore the impact of Human miRNA variants on the Human Transcriptome.
